# Advanced URL Shortener

A Flask-based URL shortener with authentication, custom short URLs, and click analytics.

## Features
- Signup & Login
- Password hashing (bcrypt)
- Session expiry
- URL shortening
- Custom short codes
- Click tracking
- User-specific history

## Tech Stack
- Flask
- SQLAlchemy
- SQLite
- Bootstrap

## Run
pip install flask flask_sqlalchemy flask-bcrypt  
python app.py  

Open:
http://127.0.0.1:5000/

## Files
app.py, models.py, auth.py, utils.py, templates/, static/, database.db, orm_sqlalchemy.ipynb
